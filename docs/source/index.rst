Welcome to pysplot's documentation!
===================================

**Pysplot** is a Python package for spatiotemporal plotting and visualizations in heliophysics.
This package contains methods for plotting spatial data and timeseries data in conjunction as well as data manipulation methods of scientific timeseries data.

This page contains the details on how to use the library, its features, and how to contribute.
